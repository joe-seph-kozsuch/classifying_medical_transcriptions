# classifying_medical_transcriptions

I made it through the first three chapters of Sebastian Raschka's Build a Large Language Model (from Scratch), but stopped there because, at the moment, I don't need to understand the inner workings of GPT. However, the underlying fundamentals, NLP, PyTorch, etc, were something I enjoyed learning and, more importantly, may actually use in my job. This purpose of this project is to solidy my learnings of tokenization and ML libraries such as PyTorch, as well as exercise other data science skills.

Dataset: 
* Source: [Medical Transcriptions](https://www.kaggle.com/datasets/tboyle10/medicaltranscriptions)
* Relevant Fields:
    * medical_specialty -> target
    * transcription -> input

Objective:
* Classify the transcriptions of medical records by specialty.
* Kaggle user ritheshsreenivasan achieved an F-1 of 0.66. That is score to beat.
